# MeetDesk

> Besprechungsnotizen für Sprint Plannings, Retros, Reviews, Dailys, Krisenprotokolle.

![MeetDesk](https://img.shields.io/badge/version-0.1.0-blue) ![Electron](https://img.shields.io/badge/Electron-29-47848F) ![React](https://img.shields.io/badge/React-18-61DAFB) ![TypeScript](https://img.shields.io/badge/TypeScript-5.3-3178C6)
## Features (Commit 1)

- ✅ 4-Panel-Layout: Sidebar · Liste · Editor · Info
- ✅ Arbeitsbereiche & Kategorien (Seed-Daten)
- ✅ Meeting-Liste mit Datumsgruppen (ein-/ausklappbar)
- ✅ Inline-Vorschau pro Meeting
- ✅ Dark / Light Mode (Token-basiert)
- ✅ Vollbild-Editor (`⌘⇧F` / `Esc`)
- ✅ Neues Meeting Modal (Geplant / Spontan)
- ✅ Aufgaben togglebar im Editor & Info-Panel
- ✅ macOS `titleBarStyle: hiddenInset` (native Traffic Lights)
